HOW TO CLONE THIS REPO OVER HTTPS
-----------------
Let's ensure our git client is properly configured with a good username and email address.

git config --global --list
git config --global user.name "First Last"
git config --global user.email "myemail@afit.edu"
git config --global http.sslverify true


Now we have completed all the setup stuff! From here on out, we get to use GIT.

To Checkout the repo, issue the following command:
cd c:\repos
git clone https://git.nykl.net/csce093/csce093.git
